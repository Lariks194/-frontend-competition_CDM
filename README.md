Или же чтобы зайти на своем компьютере и посмотреть сайт:
Простой способ запуска:
1. Скачать файлы с GitHub
Зайди на страницу проекта в GitHub
Нажми зеленую кнопку "Code"
Выбери "Download ZIP"
Распакуй архив в любую папку на компьютере

2. Запустить через Visual Studio Code
Открой Visual Studio Code
Открой папку с проектом:
Файл → Открыть папку → Выбери папку с распакованным проектом
Установи расширение Live Server:
Нажми на иконку расширений (квадратики слева)
Найди "Live Server" (автор: Ritwick Dey)
Установи его
Запусти проект:
Открой файл index.html
Нажми правой кнопкой в редакторе кода
Выбери "Open with Live Server"
Или в правом нижнем углу нажми "Go Live"
